# 🌾 AI Агро-Консультант

Интеллектуальный помощник для агробизнеса. Консультирует по регионам РФ, культурам, срокам посева/уборки и рекомендует технику из реального каталога производителя.

## 🎯 Что делает проект

| Функция | Описание |
|---------|----------|
| 🤖 **AI-консультации** | Gemini API отвечает на вопросы по агрономии и технике |
| 🚜 **Витрина техники** | Автоматически показывает характеристики, картинки и комплектации |
| 📊 **Лог панель** | Прозрачная отладка всех запросов к бэкенду |
| 🔌 **Backend на Colab** | Бесплатный хостинг через Cloudflare Tunnel |
| 🏭 **Реальный парсинг** | Сбор техники с сайта [pkyar.ru](https://pkyar.ru) |

## 🗄️ Backend (Google Colab)

Бэкенд работает на **FastAPI** и запускается в **Google Colab** с публичным доступом через **Cloudflare Tunnel**.

### Эндпоинты

| Эндпоинт | Метод | Описание |
|----------|-------|----------|
| `/health` | GET | Проверка статуса |
| `/api/status` | GET | Статус парсинга |
| `/api/chunks` | GET | Список всех товаров |
| `/api/image?url=` | GET | Прокси для картинок |
| `/api/search?q=` | GET | Поиск по товарам |

### Парсинг pkyar.ru

При запуске бэкенд автоматически:
1. Сканирует `sitemap.xml` для поиска страниц `/production/`
2. Парсит каждую страницу: название, описание, характеристики, картинку
3. Кэширует результат в `data/production.json`

## ▶️ Запуск

### 1. Установка зависимостей

```bash
npm install
```
### 2. Настройка окружения
Создайте файл .env.local:

```env
GEMINI_API_KEY=ваш_ключ_gemini
```
### 3. Запуск бэкенда (Colab)
```
Откройте Google Colab

Скопируйте код бэкенда из BackendAgroSale.ipynb

Запустите все ячейки (одна ячейка)

Скопируйте полученный URL (например, https://xxx.trycloudflare.com)

``` 
### 4. Запуск фронтенда
```bash
npm run dev
```
### 5. Подключение к бэкенду
Вставьте URL из Colab в поле Connect в интерфейсе приложения.
